This post demonstrates various Markdown formatting options.

## Headings

You can use headings to structure your content:

# Heading 1
## Heading 2
### Heading 3

## Text Formatting

You can make text **bold**, *italic*, or ***bold and italic***.

You can also use ~~strikethrough~~ text.

## Lists

### Unordered Lists

- Item one
- Item two
  - Nested item
  - Another nested item
- Item three

### Ordered Lists

1. First item
2. Second item
3. Third item

## Code

You can include `inline code` in your text.

### Code Blocks

For larger code blocks, use triple backticks:

```bash
echo "Hello, World!"
hugo server
```

## Links and Images

You can create [links like this](https://gohugo.io).

## Blockquotes

> This is a blockquote.
> It can span multiple lines.

## Tables

| Column 1 | Column 2 | Column 3 |
|----------|----------|----------|
| Data 1   | Data 2   | Data 3   |
| Data 4   | Data 5   | Data 6   |

## Conclusion

Markdown is simple yet powerful. It's perfect for writing blog posts!
